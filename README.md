# Lines of Code


## Description

This is a simple application that counts the number of lines of code in a given directories and files.


## Build & Installation

```console
> cmake -S . -B Build -DCMAKE_INSTALL_PREFIX=<installation-path>
> cmake --build Build
> cmake --install Build
```


## Usage

```console
> loc [paths]
```
